# MCU Map 文件分析工具 - 项目完成总结

## 项目概述

成功构建了一个基于 Web 的 MCU map 文件分析工具，用于分析 GCC 编译器生成的 map 文件，并通过图形化界面展示内存使用情况。

## 完成的功能

### 核心功能
- ✅ GCC map 文件解析器（支持内存配置、符号表、模块信息）
- ✅ 文件上传（拖拽/点击上传，支持 .map 文件）
- ✅ 内存使用计算（自动计算 Flash/RAM 使用量）

### 数据可视化
- ✅ 总览卡片（6个关键指标）
  - Flash/ROM 使用率
  - RAM 使用率
  - 总符号数量
  - 最大函数
  - 代码段大小
  - 数据段大小

- ✅ 图表组件
  - 内存分布饼图
  - 函数大小排名柱状图（Top 20）
  - 模块内存占用树状图
  - 模块大小排名柱状图（Top 20）

- ✅ 符号详细表格
  - 搜索功能（按符号名/文件名）
  - 排序功能（5个字段，升序/降序）
  - 分页显示（每页50条）

### 数据导出
- ✅ JSON 格式导出（完整分析数据）
- ✅ CSV 格式导出（符号列表）
